How to install

  1. Click Download installer above for the app you want.
  2. Open the downloaded ...Setup.exe. If Windows shows a "Windows protected your PC" screen, click More info → Run anyway (see note below).
  3. The installer sets the app up and adds a Desktop shortcut and Start Menu entry. No admin rights needed.
  4. Launch the app from its Desktop shortcut.

To uninstall later, use Settings → Apps → Installed apps, or the Uninstall entry in the Start Menu.

Trying the full demo

The agent drives the test app, so for the end-to-end demo install both, then:

  1. Launch Solution Test App first and leave it open.
  2. Launch Goodwill Agent and click Execute on its dashboard.
  3. Watch the agent drive the test app: Womens → Womens Coats → Better → L.

System requirements

  • OS: Windows 10 or 11 (64-bit)
  • Runtime: None — both apps are self-contained
  • Admin rights: Not required (installs per-user)

Before you run the installer

Windows SmartScreen may warn you. Because these installers aren't code-signed, Windows may show a "Windows protected your PC" screen the first time you run one. To proceed, click More info, then Run anyway. This is expected for unsigned apps and is safe here.
