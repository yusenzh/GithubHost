Goodwill Agent

A Windows desktop agent that drives the Solutions clothing-tag UI, bundled with a test app so you can try the full flow in one click.

Download for Windows GoodwillAgentBundle.zip · ~149 MB · Windows 10/11 (x64)

What's in the download

Both apps are self-contained — no .NET install required. Unzip and run.

Goodwill Agent

GoodwillAgent.exe

The agent. Runs a local HTTP receiver plus a dashboard, and drives the target app through the Windows UI Automation tree.

Solution Test App

SolutionTestApp.exe

A WPF stand-in for the Solutions DGR clothing-tag screen, so you can watch the agent work without the real software.

Quick start

  1. Download the zip above and unzip it somewhere (your Desktop is fine).
  2. Double-click start-demo.bat. Two windows open side by side.
  3. In the Goodwill Agent window, click Execute on the right-hand panel.
  4. Watch the agent drive the test app: Womens → Womens Coats → Better → L.
  5. To re-run, click New ID + Timestamp, then Execute again.

System requirements

  • OS: Windows 10 or 11 (64-bit)
  • Runtime: None — both apps are self-contained
  • Disk: ~350 MB unzipped

Before you run it

Windows SmartScreen may warn you. Because these apps aren't code-signed, Windows may show a “Windows protected your PC” screen the first time. To proceed, click More info → Run anyway. This is expected for unsigned apps.
